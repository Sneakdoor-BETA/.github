# Sneakdoor BETA

## 简介

**Sneakdoor BETA**，中文名称**测试暗门**，是一个来自中国的开发团队，主要为中国的《矩阵潜袭》玩家开发、维护各类应用。

**Sneakdoor BETA** 这一名称即来自于《矩阵潜袭》中的[一张经典卡牌](https://netrunnerdb.com/en/card/31023)。

## 项目列表

### 应用

* **[netrunner-arena](https://github.com/Sneakdoor-BETA/netrunner-arena)**：《矩阵潜袭》的网络对战平台 [Jinteki.net](https://github.com/mtgred/netrunner) 的镜像站。该项目被部署在位于中国的服务器，同时还完善了网站UI、中文卡牌文本及中文卡图等内容，以求为中国的《矩阵潜袭》玩家提供更好的网络连接质量和更好的游戏体验。
* **[netrunner-tutorial](https://github.com/Sneakdoor-BETA/netrunner-tutorial)**：《矩阵潜袭》的教学网站及人机对战平台 [Chiriboga](https://github.com/bobtheuberfish/chiriboga) 的汉化版本，将网站UI、卡牌文本及卡图进行了简体中文本地化。

### 插件

* [netrunner-arena-plugin-stats](https://github.com/Sneakdoor-BETA/netrunner-arena-plugin-stats)：将 [jnet-stats](https://github.com/multiaccess-studios/jnet-stats) 接入 *netrunner-arena* 作为数据统计面板插件，使玩家无需二次登录或下载用户数据即可在网站中直接查看自己的历史战绩统计与分析。
* [netrunner-arena-plugin-wrapped](https://github.com/Sneakdoor-BETA/netrunner-arena-plugin-wrapped)：将 [makers-eye](https://github.com/migueldlr/makers-eye) 网站中的 Jnet-wrapped 功能接入 *netrunner-arena*，使玩家可以在网站中查看自己的2025年度总结。

### 数据

* [netrunner-cards](https://github.com/Sneakdoor-BETA/netrunner-cards)：以WEBP格式保存《矩阵潜袭》中/英文卡图的数据仓库，包含各种尺寸大小的文件，并提供 Docker 脚本，可用于部署卡图图床供其他网站或应用访问。
* [netrunner-card-text-zhCN](https://github.com/Sneakdoor-BETA/netrunner-card-text-zhCN)：以JSON和CSV格式保存《矩阵潜袭》中文卡牌文本数据。
* [netrunner-database](https://github.com/Sneakdoor-BETA/netrunner-database)：以JSON格式保存《矩阵潜袭》所有卡牌数据，包含中英双语卡牌文本。
* [netrunner-arena-data](https://github.com/Sneakdoor-BETA/netrunner-arena-data)：[NoahTheDuke/netrunner-data](https://github.com/NoahTheDuke/netrunner-data) 的镜像，方便更新中文卡牌文本等源仓库不一定及时更新的内容。
