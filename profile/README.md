# Sneakdoor BETA

**Sneakdoor BETA** is a development team dedicated primarily to the Chinese *Netrunner* player community.

## Projects

### netrunner-arena

A Chinese community-maintained fork of Jinteki.net.

### netrunner-arena-plugin-stats

The Stats dashboard plugin from [jnet-stats](https://github.com/multiaccess-studios/jnet-stats) for `netrunner-arena`.

### netrunner-cards

Netrunner card images in WEBP format, can be built & deployed as a docker image for hosting.

### netrunner-card-text-zhCN

Netrunner card texts data in Simplified Chinese.

### netrunner-database

Netrunner card data in JSON format, with texts in Simplified Chinese.

### netrunner-tutorial

A localized version of [Chiriboga](https://github.com/bobtheuberfish/chiriboga), with Simplified Chinese version of card images and translated UI.
